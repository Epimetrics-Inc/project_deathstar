<subject>Subject: Amendment to Administrative Order No. 23 s. 1994 dated January 20, 1994 </subject>

<body> The Administrative Order No.23, s. 1994 on The Roles and Responsibilities of the DOH Management Team for the Health Finance Development Project (USAID Project No. 492—0446), is hereby amended to include the following revision to reflect the present set-up in the Department:

The section which reads "PROGRAM-MANAGERS: The HFDP has ﬁve program areas managed by the DOH organizational unit with functional responsibility for the technical content of the program areas. A Program Manager has been assigned responsibility for each program area. The following DOH organizational units and personnel have been designated for each program area:

The Health Policy Process program area will be managed by the Health Policy Development Staff (HPDS). The Program Manager will be the Officer—in—Charge of the HPDS." is revised as follows:

"PROGRAM MANAGERS: The HFDP has five program areas will be managed by the DOH organizational units which are functionally responsible for the technical content of each program area. A Program Manager will be designated for each program area, who is responsible for managing day-to-day activities in the program area and for achieving the outputs expected from each program area. The following DOH organizational units and personnel are designated respons1b1l1ty for each of the program areas:

The Health Policy Process program area will be managed by the Health Policy Development Staff (HPDS). The Program Manager will be the Director of the HPDS.”
As thus amended, Department Order No. 23, s. 1994 is still in effect.
